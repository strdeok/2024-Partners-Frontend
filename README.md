# 기능

### 필수 구현 사항

- URL 입력창 및 버튼
  - POST 요청을 통해 짧은 URL을 생성(Create) 합니다.
- URL 목록

  - GET 요청을 통해 원본 URL과 단축된 URL을 조회(Read) 합니다.
  - DELETE 요청을 통해 URL 요소를 삭제(Delete) 합니다.

   <br>

### 선택 구현 사항

- URL 바로 가기, URL 복사하기 등
- 스타일 관련 라이브러리(styled-components, tailwind 등) 사용 해보기
